OPFS Viewer — test fixture

Seeds OPFS for this origin with data that exercises every viewer branch. Serve over http://localhost (OPFS needs a secure context):

python3 -m http.server 8000   # then open http://localhost:8000/test/fixture.html

  • config.json — JSON (pretty-printed in preview)
  • logs/app.log — text
  • assets/img/pixel.bin — binary (has NUL → "Binary" branch)
  • big.bin — 6 MB (over inline cap → "too large" branch)
  • empty-folder/ — empty directory
  • deep/…/very-long-name….txt — long name (tooltip / truncation)
  • locked.bin — held open by a worker (locked → ⚠ in tree)
